# Contributing

This is an unofficial public derivative. Contributions must preserve the license restriction to use
with MathWorks products and services and must not imply MathWorks endorsement.

Before opening a change:

1. Keep all product workflows C-only.
2. Do not add coding clients beyond Codex, Claude Code, and manual Qwen Code configuration without a
   documented architecture decision.
3. Add tests for setup, configuration, or workflow behavior changes.
4. Run unit tests, integration tests, Ruff, mypy, and agent-document audits.
5. Preserve upstream notices on derived content and document newly introduced third-party code.

Security issues should be reported privately to the repository maintainer. Issues involving the
MathWorks product or official MCP server should also follow MathWorks support and disclosure policy.
