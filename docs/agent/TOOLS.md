# Tools

## Product CLI

- `polyspace-toolkit doctor`: read-only prerequisite discovery
- `polyspace-toolkit setup`: verified binary and automated-client setup
- `polyspace-toolkit verify`: state and readiness checks
- `polyspace-toolkit uninstall`: ownership-aware removal
- `polyspace-toolkit config validate`: public project-config validation

## Development Tools

- Pytest: behavior and integration tests
- Ruff: Python linting and import order
- mypy: strict package type checking
- Existing agent audit scripts: context, memory, module-card, and size checks
- `rg`: exact source and policy confirmation

Use real Polyspace execution only through the opt-in smoke suite or an explicitly approved user task.
