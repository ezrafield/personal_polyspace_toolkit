# User Flows

## Agent Starts A Task
1. Agent reads `AGENTS.md` or `CLAUDE.md`.
2. Agent reads `docs/agent/INDEX.md`.
3. Agent chooses relevant module cards.
4. Agent inspects targeted files.
5. Agent patches, verifies, and summarizes.

## Team Reviews Agent Work
1. Reviewer checks the final response.
2. Reviewer inspects changed files.
3. Reviewer reads the task note if the task was multi-step.
4. Reviewer confirms tests and risks.
