# Product Requirements Document

## Overview
This template helps teams create repositories that are easy for coding agents to navigate, modify, and verify.

## Goals
- Keep auto-loaded context small.
- Route agents to the right docs and files.
- Preserve task history without bloating future sessions.
- Support generated code intelligence.

## Non-Goals
- This template does not prescribe a specific application framework.
- This template does not replace human architecture decisions.
