# Task Logs

Use this folder for task-local notes when work has more than one step or needs handoff.

Recommended filename format:

```text
YYYY-MM-DD-short-task-name.md
```

Each task log should capture the goal, docs read, files inspected, commands run, whether compressed output or raw reruns were used, verification, memory extraction notes, and follow-up risk. Start from `docs/agent/TASK_LOG_TEMPLATE.md` when useful.
